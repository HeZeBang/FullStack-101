# Custom Section Divider Demo

{customExample}

This document demonstrates how to use a custom section divider defined in the frontmatter.

Instead of the default `{/**/}`, this document uses `---SECTION---` as defined in the `sectionDivider` frontmatter property.

## Features

- Custom divider pattern
- Configurable per document
- Flexible regex support
